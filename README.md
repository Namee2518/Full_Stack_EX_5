# BMI Calculator - Full Stack Experiment 5

This project is a **BMI (Body Mass Index) Calculator Web Application** created as part of a Full Stack development experiment.  
The application allows users to enter their height and weight to calculate their BMI and determine their health category.

---

## 📌 Project Description
The BMI Calculator helps users quickly calculate their Body Mass Index using a simple web interface.  
It takes user input for height and weight, calculates the BMI value, and displays the result along with the BMI category.

---

## 🚀 Features
- Simple and user-friendly interface
- Calculates Body Mass Index (BMI)
- Displays BMI category (Underweight, Normal, Overweight, Obese)
- Instant result calculation
- Responsive web design
